passgen.pl

As a Perl developer, I created a password generator script using the Crypt::RandPasswd module. The script generates passwords of a specified length, with the option to include special characters. To ensure the script is user-friendly, I added functionality to time the execution of the script, providing valuable insights into performance. I believe this script showcases my proficiency in Perl programming and my ability to work with third-party modules to deliver efficient solutions. Overall, I am proud of the value this script brings to users who need to create strong passwords quickly and easily.

https://github.com/aprunge/passgen
