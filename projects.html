passgen.pl

As a Perl developer, I created a password generator script using the Crypt::RandPasswd module. The script generates passwords of a specified length, with the option to include special characters. To ensure the script is user-friendly, I added functionality to time the execution of the script, providing valuable insights into performance. I believe this script showcases my proficiency in Perl programming and my ability to work with third-party modules to deliver efficient solutions. Overall, I am proud of the value this script brings to users who need to create strong passwords quickly and easily.

VM-QuickDeploy

VM-QuickDeploy is a shell script that simplifies the process of deploying virtual machines of any operating system. With support for netboot, users can quickly set up new VMs in seconds, and customize the deployment parameters according to their needs. The script allows users to set parameter flags for local ISO booting, memory, core count, and disk space, making it easy to create VMs with the desired specifications. This powerful tool is an excellent choice for those who need to deploy VMs quickly and efficiently, without having to go through the tedious process of setting up each VM manually.

https://github.com/aprunge/VM-QuickDeploy

py-no-cam

Py-no-cam is a program designed to enhance privacy and security by dynamically scanning for all cameras attached to the system and disabling them. Once executed, the program scans the system for any attached cameras and kicks them off of the USB bus, effectively disabling them. This ensures that nobody unwanted is able to access or use the cameras to look through and violate the user's privacy. Py-no-cam is a powerful tool that can be used to protect sensitive information and prevent unauthorized access to cameras, making it an excellent choice for those who value their privacy and security. By using py-no-cam, users can have peace of mind knowing that their cameras are protected from any unauthorized access.

https://github.com/aprunge/py-no-cam
